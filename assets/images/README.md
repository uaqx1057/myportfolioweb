# Images Folder

Place your images here:

- `profile.jpg` - Your profile photo for hero and about sections
- `project1.jpg`, `project2.jpg`, etc. - Screenshots of your projects
- `logo.png` - Your logo (optional)

## Recommended Image Sizes

- **Profile Photo**: 500x500px (square)
- **Project Screenshots**: 800x600px (landscape)
- **Logo**: 200x200px (transparent background)

## Format Recommendations

- Use **JPG** for photos
- Use **PNG** for graphics with transparency
- Use **WebP** for better compression (if browser support is not a concern)

## Optimization

Before adding images, optimize them using:
- [TinyPNG](https://tinypng.com/) - For PNG/JPG compression
- [Squoosh](https://squoosh.app/) - For advanced image optimization
- [ImageOptim](https://imageoptim.com/) - For batch processing

This will ensure your portfolio loads quickly!
